```typescript
{
  id: 'new-item',
  name: 'New Item',
  description: 'Description here',
  price: 5.99,
  image: 'ðŸ¥¨',
  category: 'churros',
  popular: true,
```

### Updating Store Information
Edit `src/app/store/page.tsx` and update the `storeInfo` object:

```typescript
const storeInfo = {
  address: 'Your Address',
  phone: 'Your Phone',
  email: 'your@email.com',
  hours: {
    monday: '7:00 AM - 9:00 PM',
    // ... other days
  }
}
```

```typescript

  <YourIcon size={20} />

```
